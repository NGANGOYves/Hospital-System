# Hospital-System
The Hospital Management System is a Flutter–Firebase app that digitizes healthcare workflows. It manages patients, medical records, appointments, and follow-ups in real time. With role-based access, digital forms, and data visualization, it improves efficiency, collaboration, and quality of care in hospitals.

# 🏥 Hospital Management System

A Flutter + Firebase-based **Hospital Management System** designed to digitalize medical workflows, improve patient care, and streamline hospital operations.  
It provides real-time patient management, medical records, and role-based dashboards for doctors, nurses, and admins.  

---

## 🚀 Features
- 🔑 Secure login with role-based access  
- 👨‍⚕️ Patient registration & medical history  
- 📅 Appointment scheduling & tracking  
- 📋 Digital medical forms (admissions, follow-ups, vaccinations, surveillance, transmissions)  
- 📊 Real-time statistics & visualization  
- 🛠 Admin dashboard for hospital management

## 🏗 Tech Stack
- **Frontend:** Flutter (mobile + web)  
- **Backend:** Firebase (Firestore, Authentication, Storage)  
- **Database:** Firestore with real-time updates  
- **Hosting:** Firebase Hosting / Flutter Web
